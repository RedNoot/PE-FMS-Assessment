## Key Interactions & Behaviors

### Class Row Click Behavior
- **Click class name or row** â†’ Navigate to Class Dashboard (view results)
- **Click [Start Assessment]** â†’ Go to Assessment Entry (pre-filled with this class)
- **Click [View Results]** â†’ Go to Class Dashboard
- **Click [â–¼ Menu]** â†’ Show action dropdown menu

### Search & Filter
- **Search:** By class name, teacher name, student name
- **Filter:** By grade, by teacher, show archived classes
- **Real-time:** Updates list as user types

### Compass Import Flow
1. User uploads Compass CSV export file
2. System detects classes in file
3. User selects which classes to import
4. System displays preview of students to be imported
5. User confirms and assigns teachers
6. Classes and students created in system
7. Success notification with next steps

### Offline Behavior
- Class list loads from local cache
- Can browse classes while offline
- Cannot create/import classes without connection
- Sync queued when connection returns
